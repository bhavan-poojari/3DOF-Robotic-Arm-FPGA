# Project Images

This folder contains images and diagrams for the 3-DOF Robotic Arm project.

## Images to Add:

### 1. FPGA Board
**Filename**: `fpga_board.jpg`
- Photo of DE10-Lite FPGA board
- Shows pin connections and setup

### 2. Robotic Arm Assembly
**Filename**: `robot_arm.jpg`
- Complete robotic arm with all servos mounted
- Shows mechanical structure and gripper

### 3. Block Diagram
**Filename**: `block_diagram.png`
- System block diagram
- Shows FPGA connections to servos
- Power supply connections

### 4. Pin Assignment Diagram
**Filename**: `pin_diagram.png`
- DE10-Lite pin mapping
- Slide switch assignments
- GPIO header connections

### 5. Circuit Connections
**Filename**: `circuit_diagram.png`
- Servo motor connections
- Power supply wiring
- Ground connections

### 6. Working Demo
**Filename**: `working_demo.jpg`
- Photo of working robotic arm
- Different positions/movements

## How to Add Images

1. Click "Add file" > "Upload files" from the main repository page
2. Navigate to this `images/` folder
3. Drag and drop your images or click to browse
4. Commit the changes

## Image Guidelines

- Use `.jpg` for photographs
- Use `.png` for diagrams and schematics
- Keep image sizes reasonable (< 5MB each)
- Use descriptive filenames
- Ensure images are clear and well-lit

## Usage in Documentation

Once uploaded, reference images in README.md using:
```markdown
![Description](images/filename.jpg)
```

Example:
```markdown
![FPGA Board Setup](images/fpga_board.jpg)
![Robotic Arm](images/robot_arm.jpg)
![Block Diagram](images/block_diagram.png)
```
